📐 二次方程的图像特征
适中
二次函数 f(x) = ax² + bx + c 的图像是一条抛物线，其形态由系数 a 决定。 当 a > 0 时，开口朝上；当 a < 0 时，开口朝下。

顶点坐标是理解抛物线的关键，可通过配方法求得：
顶点坐标公式：x = -b/(2a)，将 x 代入原函数得顶点 y 坐标。 这也是对称轴的位置。
理解顶点的意义后，我们可以通过判别式 Δ = b² - 4ac 判断抛物线与 x 轴的交点数量： Δ > 0（两个交点）、Δ = 0（一个切点）、Δ < 0（无交点）。
